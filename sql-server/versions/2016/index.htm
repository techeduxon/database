MS-SQL Server 2016 Features

SQL 2016 is packed with features such as Always Encrypted, Dynamic Data Masking, temporal tables, stretched tables, Query Store, Live Query stats and more. From our personal point of view, SQL Server 2016 is indeed an interesting release to mention. Thanks to all the testing we did with the various features of SQL Server 2016.

GeoPITS brings you the comprehensive list of SQL Server 2016 features that are supported in different editions (Enterprise Standard, Web, Express Adv, Express.)

GeoPITS Recommendations

SQL Server 2016 is a well-known and well-documented product compared to earlier versions. 2016 adds Always Encrypted, Dynamic Data Masking, Row Level Security, and temporal tables features which make it easier for you to build things to protect and track your valuable data.
